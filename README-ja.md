ブックマーク内のアクセス不能なURLをチェックしてクリーンアップする便利なChrome拡張機能です。

## 機能

- 🔍 **ブックマークスキャン**：ブックマーク内のすべてのURLがアクセス可能かどうかを自動的に検出します
- 📊 **結果表示**：スキャン結果を明確に表示し、無効なリンクは目立つスタイルでマークされます
- ☑️ **スマート選択**：無効なリンクを自動的に選択し、一括選択と個別選択をサポートします
- 💾 **データバックアップ**：クリーンアップ前に現在のブックマークデータをデスクトップに自動的にバックアップします
- 🗑️ **安全なクリーンアップ**：誤操作を防ぐため、クリーンアップ前にユーザーの二次確認が必要です
- 🌐 **多言語サポート**：中国語、英語、日本語、韓国語、フランス語、スペイン語をサポートします

## インストール方法

1. このプロジェクトをダウンロードまたはクローンしてください
2. Chromeブラウザを開き、拡張機能管理ページにアクセスします（`chrome://extensions/`）
3. 右上隅の「デベロッパーモード」を有効にします
4. 「パッケージ化されていない拡張機能を読み込む」をクリックします
5. このプロジェクトのフォルダを選択します
6. インストールが完了すると、ブラウザのツールバーに拡張機能アイコンが表示されます

## 使用方法

1. **ブックマークをスキャンする**
   - 拡張機能アイコンをクリックし、ポップアップウィンドウの「ブックマークをスキャン」ボタンをクリックします
   - スキャンが完了するまで待ちます。スキャンの進行状況と見つかった問題の数が表示されます

2. **スキャン結果を表示する**
   - スキャンが完了したら、「スキャン結果を表示」ボタンをクリックします
   - 結果ページですべてのブックマークの状態を確認できます
   - 無効なリンクは赤い枠と背景で強調表示されます

3. **無効なリンクをクリーンアップする**
   - 結果ページでは、無効なリンクが自動的に選択されます
   - クリーンアップしたくないリンクのチェックを外すことができます
   - 「選択したリンクをクリーンアップ」ボタンをクリックします
   - 確認ダイアログで操作を確認します
   - システムはまず現在のブックマークをデスクトップにバックアップし、次に選択された無効なリンクをクリーンアップします

## ファイルの説明

- `manifest.json`：拡張機能の設定ファイルで、拡張機能の基本情報とアクセス許可を定義します
- `background.js`：バックグラウンドスクリプトで、データのスキャンとクリーンアップのロジックを処理します
- `popup.html/popup.js`：拡張機能のポップアップウィンドウインターフェースと対話ロジック
- `results.html/results.js`：スキャン結果表示ページと対話ロジック
- `icons/`：拡張機能アイコンフォルダ（実際のアイコンファイルを追加する必要があります）

## 注意事項

1. 最初のスキャンには、ブックマーク内のURLの数によってはある程度の時間がかかる場合があります
2. スキャン中は拡張機能のポップアップウィンドウを閉じないでください
3. クリーンアップ操作では、デスクトップにバックアップファイルが自動的に作成されます
4. 一部のウェブサイトはセキュリティポリシーの制限により正確に検出できない場合があります
5. ブックマークの有効性を維持するには、定期的にスキャンしてクリーンアップすることをお勧めします

## ブックマークの回復方法

### ブックマークの回復方法

拡張機能は現在、ブックマークをHTML形式でバックアップしており、ChromeおよびEdgeブラウザから直接インポートすることができます。

#### Chromeブラウザブックマークの回復
1. Chromeブラウザを開き、右上隅の3つのドットメニューをクリック → ブックマーク → ブックマークマネージャー
2. ブックマークマネージャーページで、右上隅の3つのドットメニューをクリック → ブックマークと設定をインポート
3. ポップアップ表示されるインポートウィンドウで、「HTMLファイルからインポート」オプションを選択
4. 「ファイルを選択」ボタンをクリックし、以前にバックアップしたHTMLファイルを見つけて選択
5. 「インポート」ボタンをクリックして回復を完了

#### Edgeブラウザブックマークの回復
1. Edgeでお気に入りを開く：`Ctrl+Shift+O`を押すか、右上隅の3つのドットメニューをクリック → お気に入り → お気に入りの管理
2. お気に入りインターフェースの右上隅の「お気に入りをインポート」ボタンをクリック
3. ポップアップ表示されるインポートウィンドウで、「ファイルからインポート」オプションを選択
4. 「ファイルを選択」ボタンをクリックし、以前にバックアップしたHTMLファイルを見つけて選択
5. 「インポート」ボタンをクリックして回復を完了

### バックアップファイルの場所
- デフォルトでは、バックアップファイルは「ダウンロード」フォルダにダウンロードされます
- ファイル名の形式：`bookmarks_backup_日付.html`（例：`bookmarks_backup_2023-06-15.html`）
- 意図しない損失を防ぐために、重要なバックアップファイルを追加の安全な場所に保存することをお勧めします

## 技術実装

- Chrome Bookmarks APIを使用してブックマークデータを取得
- Fetch APIを使用してURLのアクセシビリティを検出
- Chrome Downloads APIを使用してバックアップファイルのダウンロードを実装
- Chrome Storage APIを使用してスキャン結果を保存
- レスポンシブUIデザインで、さまざまな画面サイズをサポート

## 開発環境

- Chromeブラウザ（Manifest V3）
- 追加の依存関係やビルドツールは必要ありません

## 多言語サポート

この拡張機能は次の言語をサポートしています：

- 中国語（簡体字）
- 英語
- 日本語
- 韓国語
- フランス語
- スペイン語

拡張機能はブラウザの言語設定に応じて表示言語を自動的に切り替えます。
